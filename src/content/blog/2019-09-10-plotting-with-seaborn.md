<div class="legacy-content"><p>Seaborn is a python library for creating plots. It is based on <a href="https://matplotlib.org">matplotlib </a> and provides a high-level interface for drawing statistical graphics.</p>&#10;<p>Seaborn integrates nicely with pandas: It operates on DataFrames and arrays and does aggregations and semantic mapping automatically, which makes it a quick, convenient option for data visualization in your data projects. One you understand the basic concepts, you can create plots really easily without using stack overflow too much.</p>&#10;<p>The basic steps for creating plots with seaborn are:</p>&#10;<ol start="0">&#10;<li>Import libraries</li>&#10;<li>Prepare the data</li>&#10;<li>[optional] Control figure aesthetics (Grids, Color etc.)</li>&#10;<li>Create plot</li>&#10;<li>[optional] Further customization of your plot, which also can overwrite the figure aesthetics defined in step 2 (title, labels, tick marks etc.)</li>&#10;<li>Show or save the plot</li>&#10;</ol>&#10;<h2 id="steps">Steps</h2>&#10;<p>Let’s walk through the steps of creating plots.</p>&#10;<h3 id="0-import-libraries">0 Import libraries</h3>&#10;<p>As mentioned above, searborn is built on top of <code>matplotlib</code>, but does not provide the full functionality. Therefore, we should always import both, <code>matplotlib </code>(with the <code>pyplot</code> module) and <code>seaborn</code>. It is likely that you will need <code>matplotlib</code> at some point (for example for changing the plot title or the tick labels and to show the plots).</p>&#10;<pre><code>import matplotlib.pyplot as plt&#10;import seaborn as sns&#10;&#10;%matplotlib inline # make sure plots are shown in a jupyter notebook. otherwise you will need plt.show()&#10;</code></pre><h3 id="1-prepare-data">1. Prepare Data</h3>&#10;<p>You can work with a pandas <code>DataFrame</code>. In general, you do not have to pre-aggregate your data. For example for count plots, seaborn will automatically calculate the counts for the count plot.</p>&#10;<h3 id="2-figure-aesthetics">2. Figure Aesthetics</h3>&#10;<p>This is an optional step. You can just go with the default configuration here, or you can define aesthetics such as grid, axis, tick marks, color and font sizes.</p>&#10;<p><strong>Axis and Grid Style</strong>: per default, you will have a dark grid. You can change between grids (<code>darkgrid</code>, <code>whitegrid</code>), no grids (<code>dark</code>, <code>white</code>) and ticks (<code>ticks</code>).</p>&#10;<pre><code>sns.set_style("white")&#10;</code></pre><p>remove the top and right axis with</p>&#10;<pre><code>sns.despine()&#10;</code></pre><p><strong>Color</strong>: Choose a <a href="https://seaborn.pydata.org/tutorial/color_palettes.html#palette-tutorial">color palette</a> and set it as the default.</p>&#10;<pre><code>sns.set_palette(palette="rocket", n_colors=3)&#10;</code></pre><p><strong>Scaling</strong>: You can choose between four contexts <code> notebook</code> (default), <code>paper</code>, <code>talk</code>, and <code>poster</code> which scales font and lines according to the use case. But you can also override some.</p>&#10;<pre><code>sns.set_context("notebook", font_scale=1.5, rc={"lines.linewidth": 2.5})&#10;</code></pre><p>Or set <strong>multiple parameters</strong> in one step with <a href="https://seaborn.pydata.org/generated/seaborn.set.html"><code>set()</code></a> e.g. <code>sns.set(context='notebook', style='white', font='Helvetica', font_scale=1, color_codes=True, rc=None))</code></p>&#10;<h3 id="3-plot">3. Plot</h3>&#10;<p>In this step, you define what kind of plot you would like. There are two important distinctions you need to understand: figure and axes</p>&#10;<p>A <strong>Figure</strong> refers to the whole figure that you see. A figure may consist of a single plot, but it is possible to have multiple subplots in the same figure. A specific sub-plot is called an <strong>Axes</strong>. There are functions that have an effect on the entire figure (figure-level functions) and some functions have an effect just on the specified subplot (axes-level-function).</p>&#10;<p>For each type of plot (distributions, categorical, relational), there is a figure-level function, which takes the kind of plot as a parameter. For example, <code>displot()</code> is the figure-level function for plotting distributions. By passing the parameter <code>kind="hist"</code>, it draws a histogram.</p>&#10;<pre><code>sns.displot(data, x, kind = "hist")&#10;</code></pre><p>Alternatively, you can choose the axes-level function <a href="https://seaborn.pydata.org/generated/seaborn.histplot.html"><code>histplot()</code></a> to plot a histogram.</p>&#10;<pre><code>sns.histplot(data, x)&#10;</code></pre><p>The table gives an overview of the figure-level functions and their axes-level functions.</p>&#10;<table>&#10;<thead>&#10;<tr>&#10;<th style="text-align:left">type of plot</th>&#10;<th style="text-align:left">figure-level function</th>&#10;<th style="text-align:left">axes-level functions</th>&#10;</tr>&#10;</thead>&#10;<tbody>&#10;<tr>&#10;<td style="text-align:left">distributions</td>&#10;<td style="text-align:left"><code>displot</code></td>&#10;<td style="text-align:left">e.g. <code>histplot</code>, <code>rugplot</code></td>&#10;</tr>&#10;<tr>&#10;<td style="text-align:left">categorical</td>&#10;<td style="text-align:left"><code>catplot</code></td>&#10;<td style="text-align:left">e.g. <code>barplot</code>, <code>boxplot</code></td>&#10;</tr>&#10;<tr>&#10;<td style="text-align:left">relational</td>&#10;<td style="text-align:left"><code>relplot</code></td>&#10;<td style="text-align:left">e.g. <code>scatterplot</code>, <code>lineplot</code></td>&#10;</tr>&#10;</tbody>&#10;</table>&#10;<p>You can use both, figure-level and axes level functions for creating a single plot figure.</p>&#10;<h3 id="4-customize">4. Customize</h3>&#10;<p>For customization, you will use some functions from&#10;<code>matplotlib.pyplot</code>. Some often used customizations are adding labels to the axis (<code>plt.xlabel</code> and <code>plt.ylabel</code>) or determining the start and end of an axis (<code>plt.ylim(0,100)</code>,<code>plt.xlim(0,10)</code>)</p>&#10;<pre><code>plt.title("A Title") # add a plot title&#10;plt.ylabel("Survived") # add a y-axis label&#10;plt.xlabel("Sex") # add an x-axis label&#10;plt.ylim(0,100) # start and end of y-axis&#10;plt.xlim(0,10) # start and end of x-axis&#10;</code></pre><h3 id="5-save-or-show">5. Save or Show</h3>&#10;<p>If you use <code>%matplotlib inline</code> in your jupyter notebook,  you will not need to call any functions for your plot to show up in the notebook. Otherwise, you will have to use <code>plt.show()</code> to show a plot and <code>plt.savefig("foo.png")</code> to save your figure.</p>&#10;<pre><code>plt.show()&#10;plt.savefig("foo.png")&#10;</code></pre><h2 id="examples-one-plot-per-figure">Examples: One Plot per figure</h2>&#10;<p>Let’s look at some examples for plots that are useful for understanding features of your dataset. id="plot-distribution-of-categorical-features-with-a-countplot">Plot distribution of categorical features with a countplot</h3>&#10;<p>Count plots automatically count the number of occurrences of each categorical feature value and displays them in a bar plot. You can just pass the dataset and the variable you want to plot as parameters. So no need to aggregate the data up front.</p>&#10;<pre><code>sns.countplot(x = 'class', data = titanic)&#10;&#10;plt.title("Count passengers by class")&#10;plt.xlabel("Class")&#10;plt.ylabel("Count nr passengers")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/0countplot.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<p>You can annotate the bars with the counts with <code>plt.annotate</code>. It takes as arguments a <code>text</code> (the counts, which equals the height of the bar), <code>xy</code> (coordinates for the annotation) and <code>data</code>. We loop over each bar (<code>plt.patches</code>) in the plot and add the annotation:</p>&#10;<pre><code>c = sns.countplot(x = 'class', data = titanic)&#10;&#10;for p in c.patches:&#10;    c.annotate(p.get_height(), &#10;                (p.get_x() + p.get_width() / 2.0, &#10;                 p.get_height()), &#10;                 ha = 'center', &#10;                 va = 'center', &#10;                 xytext = (0, 5),&#10;                 textcoords = 'offset points')&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/0countplot_annotate.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<h3 id="plot-distribution-of-numeric-features-in-a-histogram">Plot distribution of numeric features in a histogram</h3>&#10;<p>Plotting the distribution of one feature is simple: You can use <code>histplot()</code> and pass the dataset and the feature you want to plot as a parameter.</p>&#10;<pre><code>sns.histplot(data=titanic, x="age")&#10;plt.title("Age distribution")&#10;plt.xlabel("Age customer")&#10;plt.ylabel("Value")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/1histogram.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<p>You can also plot the distributions and compare across multiple groups with <code>hue</code>.</p>&#10;<pre><code>sns.histplot(data=titanic, x="age", hue="survived", multiple="stack")&#10;plt.title("Age distribution")&#10;plt.xlabel("Age customer")&#10;plt.ylabel("Value")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/1histogram2.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<h3 id="plot-relationships-between-features-with-a-scatterplot">Plot relationships between features with a scatterplot</h3>&#10;<p>You can create a simple scatter plot with 2 variables like this:</p>&#10;<pre><code>sns.scatterplot(data = titanic,&#10;               x = 'age', &#10;               y = 'fare')&#10;plt.title("Age vs. Fare")&#10;plt.xlabel("Age")&#10;plt.ylabel("Fare")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/2scatter.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<p>You can compare more than 2 variables with multiple groups using the parameters <code>hue</code> and <code>style</code>. Each will add a new group. If you specify hue and style for the same group it makes them more distinguishable.</p>&#10;<pre><code>sns.scatterplot(data = titanic,&#10;               x = 'age', &#10;               y = 'fare',&#10;               hue = 'sex',&#10;               style = 'sex')&#10;plt.title("Age vs. fare with sex")&#10;plt.xlabel("Age")&#10;plt.ylabel("Fare")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/2scatter2.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<h2 id="multiple-plots-in-one-figure">Multiple Plots in one figure</h2>&#10;<p>If you want multiple, maybe even different types of plots in one figure, you can use <code>subplots()</code> from the <code>matplotlib.pyplot</code> library. <code>plt.subplots(nrows, ncols)</code> creates a single figure instance, with nrows * ncols axes instances and returns the created figure and axes instances. The grid of the axes is returned as an <code>nrows * ncols</code> shaped numpy array. If we passed for example the parameter <code>nrows=2</code> <code>ncols=2</code>, we create a subplot with 2 rows and 2 columns. With the <code>ax</code> parameter in the axes-level functions, we specify the position of the plot. Let’s assume, we wanted to plot 2 plots side by side, we have to create a git with 1 row and 2 columns and put the plots at the correct positions.</p>&#10;<pre><code># instanciate figure and axes instances&#10;fig,axes = plt.subplots(1,2)&#10;&#10;# first plot&#10;sns.histplot(data=titanic, x="age", hue="survived", multiple="stack",ax=axes[0]) #first position&#10;plt.xlabel("Age")&#10;plt.ylabel("Counts")&#10;&#10;# second plot &#10;sns.scatterplot(data = titanic,&#10;               x = 'age', &#10;               y = 'fare',&#10;               hue = 'survived',&#10;               style = 'survived',ax=axes[1]) #second position&#10;plt.xlabel("Age")&#10;plt.ylabel("Fare")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/3subplot.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<p>You can plot as many axes as you want into one figure. You can pass the position of the plot as <code>ax = 'axes[row, column]</code>. If you add more plots, you may need to increase the size of the figure so the plots are still readable</p>&#10;<pre><code>fig,axes = plt.subplots(2,2) &#10;&#10;# increase figure size&#10;fig.set_figheight(10) &#10;fig.set_figwidth(10)&#10;&#10;sns.histplot(data=titanic, x="age", ax=axes[0,0])&#10;sns.histplot(data=titanic, x="fare", ax=axes[0,1])&#10;sns.histplot(data=titanic, x="pclass", ax=axes[1,0])&#10;sns.histplot(data=titanic, x="survived", ax=axes[1,1])&#10;&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/3subplot2.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<h2 id="facetgrids">Facetgrids</h2>&#10;<p>The figure-level functions <code>relplot</code>, <code>catplot()</code> and <code>displot</code> are built on a <code>FacetGrid</code>. This means that it is easy to add faceting variables to visualize higher-dimensional relationships:</p>&#10;<pre><code>sns.displot(data=titanic, x="age", hue="survived", col="survived")&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/4distplot.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<p>The <code>FacetGrid</code> class is useful when you want to visualize the distribution of a feature or the relationship between multiple features separately within subsets of your dataset. You may use three parameters to specify the dimensions <code>row</code>, <code>col</code>, and <code>hue</code>. <code>row</code> and <code>col</code> correspond to the array of axes. You can use <code>hue</code> to differentiate between different levels as you did in the scatter plot examples above.</p>&#10;<pre><code>with sns.axes_style("white"):&#10;    g = sns.FacetGrid(titanic, row="sex", col="pclass", margin_titles=True, height=2.5)&#10;g.map(sns.barplot, "survived", "fare")&#10;g.fig.subplots_adjust(wspace=.02, hspace=.02)&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/4distplo2.png" alt="Illustration for Plotting with Seaborn"></p>&#10;<h2 id="quickly-explore-numerical-features-with-a-pairplot">Quickly explore numerical features with a <code>pairplot</code></h2>&#10;<p>One quick and easy way to start exploring a dataset is with <code>sns.pairplot()</code>. It automatically draws a scatter plot for the correlations between all numeric variables. The diagonal shows the distribution of each numeric feature.</p>&#10;<p>For example for the first 4 columns in the titanic dataset, we will get a 3 x 3 grid, because there are 3 numeric variables in the dataset in the first 4 columns.</p>&#10;<pre><code>sns.pairplot(titanic.iloc[:,:4])&#10;</code></pre><p><img loading="lazy" decoding="async" src="/post/2019-09-10-seaborn/5pairplot.png" alt="Illustration for Plotting with Seaborn"></p></div>
