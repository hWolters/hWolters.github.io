You have to apply <code>cell_spec</code> to a <code>data.frame</code> and not <code>knitr_kable</code> object. First you have to create a new column in your <code>data.frame</code> with <code>cell_spec</code> for example by using the <code>mutate</code> function from the <code>dplyr</code> package. Second, apply <code>kable(escape = FALSE)</code>. The <code>cell_spec</code> function creates pure html code. That’s why you need to pass the additional argument <code>escape = FALSE</code>. Currently, <code>cell_spec</code> supports features including bold, italic, monospace, text color, background color, align, font size and rotation angle.</p>&#10;<p>Here is an example how you could
